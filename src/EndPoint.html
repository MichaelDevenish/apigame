EndPoint is a Roguelike Text Based Retro API RPG or RTBRAPIRPG all the commands in this game are given through the website URL bar. Once you copy your API key into the URL bar you can begin to give commands. The commands are:


  • /help - Opens a help Screen
  • /attack/[Enemy Name] - attacks the enemy
  • /equip/[weapon or armour] - equip armour or weapon
  • /go/[Direction] - go in a direction north, south, east or west
  • /killme - Kills you
